🌍 Base Système

Ces paramètres affectent tous les modules simultanément.

Utilise : rgb(from var(--c) r g b / alpha)
MIX

1. Le Color-Mixer

Choisis l'espace colorimétrique d'interpolation. Remarque comme `srgb` crée des couleurs ternes par rapport à `oklab` ou `oklch`.

color-mix(in oklab, var(--base) 50%, var(--mix))
50%
OKLCH

2. Modificateurs RCS (oklch)

Altère la couleur en décomposant ses canaux. Idéal pour générer des thèmes dynamiques.

oklch(from var(--base) calc(l + L) calc(c + C) calc(h + H))
0
0
0deg
FILTER

3. Effets de Filtres (GPU)

Contrairement à RCS, `filter` agit sur le rendu final des pixels via la carte graphique. Parfait pour les interactions au `:hover` ou les images.

filter: hue-rotate(...) saturate(...) contrast(...) invert(...)

TEXTE AUTO

4. Auto-Contraste

Génère un texte Blanc ou Noir de façon à garantir une accessibilité parfaite selon la luminosité de la couleur de fond.

color: color-contrast(var(--base) vs white, black)
⚠️ Tech Note : La spécification native color-contrast() n'étant pas encore supportée par Chrome/Firefox, ce bloc utilise un algorithme JS (Polyfill) qui calcule la luminance relative YIQ pour simuler la fonction CSS finale ! Change la couleur de base ou son opacité pour tester.
